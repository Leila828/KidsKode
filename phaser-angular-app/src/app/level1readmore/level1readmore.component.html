Block Based Coding

Block-based coding simplifies the process of computer programming.

Rather than writing out the code, students can drag and

drop coding blocks that represent code. Block-based

coding uses interlocking, graphical blocks

so students don’t need to write. The

benefit of using block-basedcoding is

that it helps younger students learn the basics

of programming without having to worry about

the messy syntax that often accompanies code.
